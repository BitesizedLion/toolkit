## Description

[The Collective Awareness to UXO (CAT-UXO)](https://cat-uxo.com/) app is a go-to repository for many professionals working in the explosive ordnance disposal (EOD) space. It has categorized hundreds of unique models of EO, including all land service ammunition categories. Each entry in CAT-UXOâ€™s database provides an image of the item of EO and a brief description. Each category can also be filtered by further sub-categories, and its country of origin, country of use, weight, and dimensions.

CAT-UXO is based in the United Kingdom with contributors adding information from all regions of the world.
